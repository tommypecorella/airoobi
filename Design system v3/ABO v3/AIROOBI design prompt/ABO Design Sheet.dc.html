AIROOBI · Theme v3 skin · 18 Jul 2026

ABO — Token & Component Sheet

Restyle of the AIROOBI Back Office as a skin, not a rewrite: a token layer + component CSS mapped onto the existing classes (admin-sidebar, admin-nav, admin-card, admin-table/adm-tbl, tag, ab-btn, ct-modal, abo-todo-item) — IDs, JS and flows untouched. Mobile-first at 390px, full light + dark from the theme-v3.css tokens already in production. Table strategy: horizontal scroll + sticky first column (your pick). Red primary used sparingly; no gold, no purple, no gambling/trading visuals.
